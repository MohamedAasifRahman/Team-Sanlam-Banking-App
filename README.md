# 🏦 Sanlam Banking App

> A collaborative Java backend project simulating real-world banking software development - built to industry standards as part of our academic journey.

---

## 📌 About This Repository

This repository contains the source code for the **Sanlam Banking App**, a Java-based backend application developed by our team as part of our academic software development programme. The project is designed to mirror professional banking software workflows, applying industry-level coding practices and clean architecture principles.

---

## 🎯 Project Goals

- Build a fully functional **Java backend banking application**
- Apply **object-oriented programming** and software design principles
- Follow **industry-style development workflows** using Git & GitHub
- Write clean, well-documented, and maintainable code
- Develop strong habits around **version control, code review, and collaboration**
- Strengthen our profiles as **industry-ready software developers**
